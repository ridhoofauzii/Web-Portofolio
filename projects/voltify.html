Web App · Fuzzy Logic

⚡ Voltify

Sistem pendukung keputusan untuk monitoring & manajemen konsumsi listrik rumah tangga berbasis web, menggunakan metode Fuzzy Logic.

👥 Tim: 3 orang (project kuliah) 🎓 Peran: Web Developer 📄 Terbit: Jurnal JATI Vol. 10 No. 2 (2026)

📖 Tentang Project

Konsumsi listrik rumah tangga terus meningkat, namun meteran konvensional hanya menampilkan total pemakaian tanpa rincian per perangkat. Hal ini menyulitkan pengguna mengidentifikasi perangkat boros energi.

Voltify hadir sebagai aplikasi web yang memanfaatkan Fuzzy Logic untuk mengklasifikasikan tingkat konsumsi energi perangkat. Pengguna memasukkan daya perangkat (Watt) dan durasi penggunaan (jam/hari), lalu sistem menghitung konsumsi energi (kWh), biaya harian, serta mengklasifikasikannya ke kategori Hemat, Normal, atau Boros.

✨ Fitur Utama

  • Klasifikasi Fuzzy Logic — menilai konsumsi perangkat jadi Hemat / Normal / Boros beserta tingkat keyakinan.
  • Hitung kWh & biaya — kalkulasi konsumsi energi dan estimasi biaya harian/bulanan.
  • Dashboard statistik — ringkasan mingguan dengan grafik konsumsi.
  • Estimasi biaya — proyeksi total biaya listrik.
