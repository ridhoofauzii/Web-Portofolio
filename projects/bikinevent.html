🎟️ Bikin Event

Sistem manajemen event dan sertifikat digital berbasis web yang komprehensif, dari perencanaan acara hingga distribusi sertifikat.

👥 Tim: Kolaborasi 🌐 Platform: Web

📖 Tentang Project

Bikin Event adalah sistem manajemen event dan sertifikat berbasis web yang dirancang untuk memudahkan pengelolaan acara dari tahap perencanaan hingga distribusi sertifikat.

Platform ini memungkinkan administrator mengelola event secara efisien dengan analytics real-time, export data, dan generator sertifikat otomatis. Peserta dapat mendaftar ke berbagai event dan mengunduh sertifikat digital mereka.

✨ Fitur Utama

  • Dashboard Admin — ringkasan statistik real-time dengan visualisasi data.
  • Kelola Event — CRUD event dengan form komprehensif dan upload gambar.
  • Kelola Peserta — manajemen data peserta dengan pencarian dan filter.
  • Laporan & Analytics — dashboard analytics dengan grafik tren bulanan.
  • Generator Sertifikat — sertifikat otomatis dengan template kustomisasi.
  • Export Data — export ke Excel dan PDF untuk laporan.
  • Dashboard Peserta — overview personal dengan event yang diikuti.
  • Download Sertifikat — akses dan unduh sertifikat digital PDF.
